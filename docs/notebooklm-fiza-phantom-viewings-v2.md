## Core Features (What to Highlight)

### 1. Bitcoin Wallet Sign-In
- Users can sign in with their Bitcoin wallet instead of email/password
- No more passwords to forget. No more “forgot password” flows.
- Web3-native, secure, and familiar to crypto users.

### 2. Passport Trust Score
- Every user gets a portable Trust Score from 0–1000
- Tiers: Platinum (850+), Gold (700+), Silver (500+), Bronze (300+), Unrated (0)
- Score tracks: completed bookings, no-shows, disputes, punctuality rate
- Score travels with the user across every Pabandi business
- Bad actors can’t create new accounts to escape their history

### 3. Earnings & Staking (Hibah Model)
- Users earn **PabPoints** for being reliable: showing up on time, completing bookings, earning good reviews
- PabPoints are **not crypto staking with interest** — they are a **Hibah** (gift), Sharia-compliant, no ribā, no gharar, no maysir
- **Redeem path:** PabPoints → Solana (SOL) → withdraw to bank/payment provider
- This means reliable users literally get paid for being reliable: real money, not just app points
- No lock-up periods. Earn daily, redeem anytime

### 4. AI No-Show Prediction
- Agentic AI predicts no-show probability before booking is confirmed
- Dynamic deposits calculated in real-time based on risk
- High-risk users prompted for upfront payment automatically
- Reduces no-shows by 60%+

### 5. Dual-Engine Flywheel
- **Consumer App:** Booking platform for salons, clinics, live sellers in Pakistan
- **Global API:** Zero-knowledge fraud prevention API for Shopify, TikTok Shop, Daraz
- Consumer behavior data feeds the API; API revenue funds consumer growth

### 6. WhatsApp Automation
- Automated booking confirmations via Meta Cloud API
- 24-hour reminders
- Owner alerts and post-booking review requests
- All in Roman Urdu + English mix

### 7. Payments Ecosystem
- **Stripe:** US cards, Apple Pay, Google Pay
- **Safepay:** Pakistani cards, JazzCash
- **Solana Web3:** Phantom/Solflare wallet integration
- Escrow releases automated post-visit

### 8. Security & Compliance
- ISO 27001 ISMS compliant
- Sharia-compliant loyalty model (no ribā, no gharar, no maysir)
- KYC/AML required before wallet withdrawals
- Zero-knowledge dietary passport (E2EE)
