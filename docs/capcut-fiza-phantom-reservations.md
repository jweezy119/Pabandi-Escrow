## Quick Assembly Order (For Speed)

1. Import all 4 scene visuals/images into timeline
2. Set each clip duration (Scene 1: 7s, Scene 2: 7s, Scene 3: 8s, Scene 4: 8s)
3. Add BGM track, split into 4 segments matching scenes
4. Add voiceover track (Fiza + infomercial VO)
5. Add SFX on separate audio layer
6. Add text overlays one by one, copy-paste from cheat sheet above
7. Apply transitions (cut/zoom blur/flash)
8. Preview full timeline once
9. Export 1080x1920

Total estimated assembly time: 15-20 minutes in CapCut Pro.
